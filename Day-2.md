## Python Data Types

The trainer explained the basic data types used in Python.

### Integer (int)
Stores whole numbers.

**Example:** Age = 21

### Float (float)
Stores decimal values.

**Example:** Salary = 55000.75

### String (str)
Stores text.

**Example:** Name = "Aman"

### Boolean (bool)
Stores only two values: True or False.

**Example:** is_Placed = True

**Importance:**
Understanding data types is essential because Python performs different operations depending on the type of data.

We learned different data structures used to store collections of data.

### List

## Pandas Data Structures

### Series

A Series is a one-dimensional array that stores data in a single column.

### DataFrame

A DataFrame is a two-dimensional table similar to an Excel spreadsheet. It is the most commonly used data structure for data analysis.
