### Summary

Today, I learned about **Wrapper Feature Selection**, a technique that uses machine learning algorithms to select the best subset of features based on model performance. The trainer explained how wrapper methods evaluate different feature combinations to improve prediction accuracy while reducing unnecessary features.

During the practical session, we implemented **Forward Selection**, **Backward Elimination**, **Recursive Feature Elimination (RFE)**, and **Recursive Feature Elimination with Cross Validation (RFECV)** using **Logistic Regression** and **Decision Tree Classifier**. We compared the features selected by each method and observed how different wrapper techniques choose the most relevant features for building an efficient machine learning model.

### Key Learning
- Understanding the concept of Wrapper Feature Selection.
- Applying Forward Selection to add important features step by step.
- Using Backward Elimination to remove less important features.
- Implementing Recursive Feature Elimination (RFE) for feature ranking.
- Learning how RFECV automatically selects the optimal number of features.
- Comparing different wrapper methods based on model accuracy.
- Improving model performance by selecting the most relevant features.
