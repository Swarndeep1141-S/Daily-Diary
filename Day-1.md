## Introduction to AI & ML

Today was the first day of the AI & Machine Learning training program. The trainer introduced the basic concepts of Artificial Intelligence (AI) and Machine Learning (ML).

Artificial Intelligence is a branch of computer science that enables machines to perform tasks requiring human intelligence, such as learning, reasoning, and decision-making.

Machine Learning is a subset of AI that allows computers to learn from data and make predictions without being explicitly programmed.

---

## Data Exploration

Data Exploration is the process of examining and understanding a dataset before building a machine learning model.

It helps to:
- Understand the dataset
- Find missing values
- Detect duplicate records
- Identify outliers
- Discover relationships between variables

### Real-Life Example

For a House Price Prediction project, data exploration is used to analyze features such as:
- City
- Area (sq. ft.)
- Number of Bedrooms
- Number of Bathrooms
- House Age

This helps determine which features have the greatest impact on house prices.
